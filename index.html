Considerations About the Program

I made this dedicated program for watching TNT on Molotov.TV named Playtov. It comes with another program, if you have a laptop PC and Bluetooth speakers, to enable both. There's a parameters file under same folder of the program if you have a error message, it need to set the parameter to True or False, it's made if first device is the capture device. To note, it ask to set a audio quality of 16 bits and 44100 Hz format for both audio cards, and enable both material acceleration and enhanced audio, under audio settings, but there's also another parameter to disable the program to not automatically launching.
